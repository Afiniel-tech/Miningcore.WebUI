how to setup your miner to this pool.

Getting started

To get started mining and use this pool you need the following

  • {{ coinName }} Wallet address
  • crypto mining software that supports the {{ coinName }} coin and aglorithm {{ aglorithm }}
  • hardware to run it on. This can be your home PC, mining rig, ASIC miner or cloud mining.

{{ coinName }} Wallet address

A wallet address is needed to payout you shares mined at this pool server.
When the total mined value is past the payout threshold, we will send your coin to this wallet address.


crypto mining software

To mine at this pool you can use any miner supporting the {{ aglorithm }} aglorithm or {{ coinName }} coin.
Use an search engine and search for "{{ coinName }} miner software". download the miner software and configure your crypto miner.

Where:

  • POOL STRATUM ADDRESS AND PORT - one off the stratum addresses above in the Pool Configuration section depending on the difficuty you want
  • YOUR_WALLET_ADDRESS - your valid {{ coinName }} wallet address
  • WORKERNAME - an optional workername can be used to identify the miner or RIG
  • PASSWORD - use x or leave it blank

    • Optional:
  • STATIC DIFFICULTY - to mine with a static (fixed) difficulty simply use d=xxx as password in your miner configuration where xxx denotes your preferred difficulty.

For amd cards These environment variables should be set to for the driver to allow max mem allocation from the gpu(s). make sure you add this at the top of your .bat file:

AMD example of bat file:
		
set GPU_MAX_ALLOC_PERCENT=100
set GPU_SINGLE_ALLOC_PERCENT=100
set GPU_MAX_HEAP_SIZE=100
set GPU_USE_SYNC_OBJECTS=1

myminer -a ethash -o stratum+tcp://MyStratumAddress:PORT -u wallet.workername -p x,d=1
